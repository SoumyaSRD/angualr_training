Setup Steps

Step 1: Install Node.js (includes NPM)

  1. Go to nodejs.org
  2. Download the LTS version (recommended)
  3. Install it like any other program
  4. Open terminal/command prompt

Step 2: Check Installation


# Check Node.js version
node --version

# Check NPM version
npm --version
