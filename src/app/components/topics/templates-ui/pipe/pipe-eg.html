Pipes in Angular are used to transform and format data directly within templates. They provide a clean and declarative way to modify how values are displayed without changing the underlying data. Pipes help keep component logic simple by moving display-related transformations into reusable functions.

Angular includes several built-in pipes for common data transformations such as formatting dates, currencies, numbers, JSON objects, and text casing. Developers can also create custom pipes to implement application-specific formatting or transformation logic. Pipes are applied in templates using the pipe (|) operator, making templates more readable and expressive.

Why Use Pipes?

  • Cleaner Templates: Simplifies data transformation directly in HTML.
  • Reusable Logic: Write once and reuse across multiple components.
  • Separation of Concerns: Keeps formatting logic separate from business logic.
  • Improved Readability: Makes templates easier to understand.

Common Built-in Pipes

  • DatePipe: Formats dates according to locale settings.
  • CurrencyPipe: Formats numbers as currency values.
  • DecimalPipe: Formats numeric values with decimal precision.
  • UpperCasePipe / LowerCasePipe: Converts text casing.
  • JsonPipe: Displays objects in JSON format for debugging.
  • AsyncPipe: Automatically subscribes to Observables or Promises and updates values when data changes.

Using Pipes in Templates

Pipes are applied using the | symbol followed by the pipe name. 


      <p>{{ username | uppercase }}</p>
      <p>{{ day | date:'mediumDate' }}</p>

Custom Pipes

Developers can create custom pipes when built-in pipes are not sufficient. Custom pipes are created using the @Pipe decorator and implement the PipeTransform interface. They allow custom transformation logic to be reused across templates.

@for (item of employees | customFilter:searchTermNorm; track item.id; let i = $index) { }
ID Name Department Designation Salary
{{ item.id }} {{ item.name }} {{ item.department }} {{ item.designation }} {{ item.salary | currency:'INR':'symbol':'1.0-0' }}

@for (item of employees | customFilter:searchTerm:['name','department','designation']; track item.id; let i = $index) { }
ID Name Department Designation Salary
{{ item.id }} {{ item.name }} {{ item.department }} {{ item.designation }} {{ item.salary | currency:'INR':'symbol':'1.0-0' }}

Pure vs Impure Pipes

  • Pure Pipes: Execute only when input values change (default behavior, better performance).
  • Impure Pipes: Execute on every change detection cycle; used when data changes frequently without reference updates.

Summary

Pipes enhance Angular templates by providing an efficient way to transform and format data. They improve code readability, encourage reuse, and help maintain a clean separation between presentation and logic.
